# SmartPoint

User Management Interface (Vue 3 + PrimeVue + Pinia)

---

## Stack

- **Vue 3** — Composition API (`<script setup>`)
- **Vite** — modern dev server & bundler
- **PrimeVue** — UI components (Menubar, DataTable, Button)
- **PrimeIcons** — icon set for controls
- **PrimeFlex** — utility-first layout system
- **Pinia** — state management (favorites store)
- **Axios** — API requests
- **LocalStorage** — persistent favorites saving
- **ESLint + Prettier** — formatting & code quality

## API Used

https://jsonplaceholder.typicode.com/users

---

## Project Setup

```bash
# 1. Clone repository
git clone

# 2. Install dependencies
npm install

# 3. Run in development mode (http://localhost:5173)
npm run dev

# 4. Build for production
npm run build

# 5. Preview built app locally
npm run preview
```
